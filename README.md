## Story

wp-detective started as a Chrome extension that detects WordPress sites and surfaces admin shortcuts, hosting info, and developer tools in the browser toolbar. Through the v0.x line, it grew:

- Safari support via a companion Xcode project
- A React popup architecture, Safari companion, Site Information panel, and Block Inspector — substantial contributions from [Fabian Kägy](https://github.com/fabiankaegy)
- Host detection for the major managed-WordPress platforms (WP Engine, WordPress VIP, Pantheon, Kinsta, Flywheel, Cloudways, Pressable, WordPress.com)
- A developer-tools surface (mobile preview, cache bypass, block highlighting, Query Monitor toggle)

At v0.8 the project moved to the WordPress org to become the official WordPress browser extension, with initial directory releases (Chrome Web Store and Safari / Mac App Store) targeted for v1.0 and Firefox / Edge to follow.

This repository remains as the historical home of the v0.x line. The full commit history is intact.
